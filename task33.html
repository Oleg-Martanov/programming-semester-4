Теорема о функции распределения случайной величины

Формулировка

Функция распределения F(x) = P{ξ ≤ x} обладает следующими свойствами:

  1. F(x) не убывает,
  2. limx→-∞ F(x) = 0, limx→∞ F(x) = 1
  3. F(x) непрерывна справа, P{ξ < x} = F(x - 0) = limx↑yF(y)

Доказательство

1. Если x < y, то {ξ ≤ y } ⊇ {ξ ≤ x } и так как вероятность - монотонная функция:

F(x) = P{ξ ≤ x} ≤ P{ξ ≤ y} = F(y)

2. Из равенства Ω = ∪n≥1 (Cn\Cn-1), где Cn = {ω : -n < ξ ≤ n}, C0 = ∅, получаем, что

1 = P{Ω} = Σn≥1P{Cn\Cn-1} = limN→∞ ΣNn=1 P{Cn\Cn-1} =
= P {Cn} = limn→∞ (F(N) - F(-N)).

Теперь из того, что F(x) ∈ [0,1] при всех x, следуют равенства

limx→-∞ F(-∞)=0, limx→-∞ F(∞)=1,

3. Рассмотрим последовательность непересекающихся множеств

Bn = (ω: x + 1/(n+1) < ξ(ω) ≤ x + 1/n), n = 1, 2, …

Тогда {ω: x < ξ ≤ x+1} = ∪n≥1 Bn и

F(x+1) - F(x) = P{x < ξ ≤ x+1} = P{∪n≥1 Bn} =
= Σn≥1 P{Bn} = Σn≥1 (F( x + 1/n) - F(x + 1/(n+1))) =
= limN→∞ ΣNn=1 (F( x + 1/n) - F(x + 1/(n+1))) =
= limN→∞ (F( x + 1) - F(x + 1/(N+1))).

Следовательно

F(x) = limN→∞ F(x + 1/(N+1)) = F(x + 0) = limy↓x F(y).

Аналогично, рассматривая непересекающиеся множества

A1 = {ω: ξ ≤ x -1}, An = {ω: x - 1/(n-1) < ξ ≤ x - 1/n}, n≥2,

для которых {ω: ξ < x} = ∪n≥1 An, находим

P{ξ < x} = Σn≥1 P{An} =
= F(x - 1) + Σn≥2 (F(x - 1/n) - F(x - 1/(n-1))) =
= limN→∞ (F(x - 1) + ΣNn=2 (F(x - 1/n) - F(x - 1/(n-1)))) =
= limN→∞ F( x - 1/N) = limy↑x F(y) = F(x - 0).
